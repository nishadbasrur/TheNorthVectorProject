# ADR-0088: Use Knowledge Consolidation and Archival for Stale or Completed Work

Status: Accepted

North Vector will consolidate and archive stale, completed, inactive, or superseded work so the active system remains clear.

Archival should preserve important history without cluttering active planning, retrieval, and decision workflows.

The system should distinguish active knowledge from archived knowledge.

Outcome: North Vector maintains clarity while preserving useful historical context.
