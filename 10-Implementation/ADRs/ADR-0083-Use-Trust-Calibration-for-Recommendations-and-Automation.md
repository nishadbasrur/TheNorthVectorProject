# ADR-0083: Use Trust Calibration for Recommendations and Automation

Status: Accepted

North Vector will calibrate trust based on confidence, evidence, uncertainty, risk level, and historical performance.

The system should not present weak recommendations with the same force as highly supported recommendations.

Trust calibration should affect wording, required review, automation eligibility, and escalation behavior.

Outcome: North Vector becomes more honest, safer, and more useful by matching confidence to presentation and action level.
