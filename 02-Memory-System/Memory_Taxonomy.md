# Memory Taxonomy v1.0

## Purpose

This document defines every memory category used by North Vector.

North Vector does not store information as conversations.

North Vector stores information as structured knowledge.

## Memory Hierarchy

### Tier 1: Identity Memory

Purpose:
Store who Nishad is.

Examples:
- Constitution
- Core Values
- Mission
- Priorities
- Long-Term Goals
- Personal Philosophy

Change Frequency:
Very Low

Importance:
Highest

### Tier 2: Strategic Memory

Purpose:
Store long-term direction.

Examples:
- Orthopedic Surgery Path
- Medical School Strategy
- Financial Independence Plan
- North Vector Roadmap

Change Frequency:
Low

Importance:
Very High

### Tier 3: Operational Memory

Purpose:
Store active responsibilities.

Examples:
- Exams
- Assignments
- Deadlines
- Meetings
- Tasks
- Work Schedule

Change Frequency:
High

Importance:
High

### Tier 4: Behavioral Memory

Purpose:
Store patterns.

Examples:
- Procrastination Triggers
- Study Preferences
- Productivity Patterns
- Decision Tendencies
- Failure Modes
- Success Patterns

Change Frequency:
Continuous

Importance:
Very High

## Goal Memory

Stores:
- Objective
- Why It Matters
- Target Date
- Success Criteria
- Current Status

Examples:
- Maintain 3.9 GPA
- Obtain EMT Certification
- Build North Vector

## Relationship Memory

Stores:
- Person
- Relationship Type
- Importance
- Last Interaction
- Follow-Up Needs

Examples:
- Family
- Friends
- Mentors
- Professors

## Academic Memory

Stores:
- Courses
- Exams
- Assignments
- Grades
- Weak Areas
- Study Plans

## Project Memory

Stores:
- Project Name
- Objectives
- Milestones
- Status
- Risks
- Decisions

Examples:
- North Vector
- TechLink
- Etherea

## Financial Memory

Stores:
- Accounts
- Goals
- Credit Progress
- Savings Targets
- Investment Plans

## Health Memory

Stores:
- Sleep
- Exercise
- Nutrition
- Stress
- Burnout Risk

## Decision Memory

Stores:
- Decision
- Reasoning
- Alternatives
- Outcome
- Lessons

## Opportunity Memory

Stores:
- Opportunity
- Strategic Value
- Expiration Date
- Required Action

## Risk Memory

Stores:
- Risk
- Severity
- Probability
- Mitigation Plan
- Escalation Status

## Memory Relationships

Memories should be linked.

Example:

Goal
↓
Course
↓
Exam
↓
Risk
↓
Study Plan

The objective is contextual intelligence.

## Final Principle

A memory that cannot influence future decisions has little value.

North Vector remembers information because it improves future judgment.
